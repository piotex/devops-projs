**Source code:**
[https://spring.io/guides/gs/spring-boot](https://spring.io/guides/gs/spring-boot)

**Install java 17:**
```
sudo apt update
sudo apt install openjdk-17-jdk -y
```

**...**
```
mvn clean install
java -jar target/spring-boot-complete-0.0.1-SNAPSHOT.jar
```

**Start app:**
```
./gradlew bootRun
```
